Лекции 2&3 — Асимптотични нотации и Рекурентни уравнения

Напредък

0 от 11 теми изучени

Θ-нотация — Точна асимптотика

1
Точна дефиниция
Определение — Θ-нотация

Нека \(g: \mathbb{N} \to \mathbb{R}^+\). Дефинираме:

\[\Theta(g(n)) = \{ f \mid \exists c_1 > 0,\, c_2 > 0,\, n_0 \in \mathbb{N} : \forall n \geq n_0 : c_1 g(n) \leq f(n) \leq c_2 g(n) \}\]

Казваме \(f(n) = \Theta(g(n))\) (по злоупотреба с нотацията, вместо \(f \in \Theta(g(n))\)) и четем "f е тета на g".

Лесно обяснение

Θ описва точната асимптотична наредба. Ако \(f(n) = \Theta(g(n))\), то \(f\) и \(g\) растат с еднаква скорост — само константни фактори ги различават. Θ е едновременно горна И долна граница.

Аналогия: \(\Theta\) е като знак "=" за функции, \(O\) е като "≤", \(\Omega\) е като "≥".

Визуализация
↑ f(n) c₂·g(n) ___/‾‾‾‾‾‾‾‾ ← горна граница / f(n) /~~~~~~~~~~~~~~ ← между двете граници \___ c₁·g(n) ‾‾‾‾‾‾‾‾ ← долна граница → n n₀
Теорема 16: Еквивалентна характеризация

\(f(n) = \Theta(g(n))\) тогава и само тогава когато \(f(n) = O(g(n))\) и \(f(n) = \Omega(g(n))\).

Примери
  • \(3n^2 + 5n + 7 = \Theta(n^2)\) — с \(c_1=3, c_2=15, n_0=2\)
  • \(n \log n + n = \Theta(n \log n)\)
  • \(2^n \neq \Theta(n^{100})\) — различни скорости на нарастване
За изпит
  • Три константи: \(c_1, c_2, n_0\). Съществуват. За всяко \(n \geq n_0\).
  • \(f = \Theta(g)\) ⟺ \(f = O(g)\) AND \(f = \Omega(g)\).
  • Θ е рефлексивна и симетрична и транзитивна (релация на еквивалентност).
Изпитен въпрос

Дефинирайте Θ-нотацията. Докажете, че \(f(n) = \Theta(g(n)) \Leftrightarrow f(n) = O(g(n))\) и \(f(n) = \Omega(g(n))\).

O-нотация — Горна асимптотична граница

2
Точна дефиниция
Определение — O-нотация (Big-O)

Нека \(g: \mathbb{N} \to \mathbb{R}^+\). Дефинираме:

\[O(g(n)) = \{ f \mid \exists c > 0,\, n_0 \in \mathbb{N} : \forall n \geq n_0 : f(n) \leq c \cdot g(n) \}\]

Казваме \(f(n) = O(g(n))\) и четем "f е big-O на g". \(g\) е асимптотична горна граница на \(f\).

Теорема 17: Свойства на O
  • Ако \(f(n) = \Theta(g(n))\), то \(f(n) = O(g(n))\).
  • \(O(g(n))\) е транзитивна: \(f=O(g)\) и \(g=O(h)\) ⟹ \(f=O(h)\).
  • Ако \(\lim_{n\to\infty} f(n)/g(n)\) съществува и е крайно, то \(f(n) = O(g(n))\).
Примери
  • \(n^2 = O(n^3)\) — с \(c=1, n_0=1\)
  • \(n \log n = O(n^2)\)
  • \(5n + 3 = O(n)\) — с \(c=8, n_0=1\)
  • \(2^n \neq O(n^{100})\) — защото \(2^n/n^{100} \to \infty\)
За изпит
  • O е горна граница — може да е много по-голяма от f!
  • Само две константи: \(c, n_0\). Съществуват.
  • O е транзитивна, рефлексивна, но НЕ симетрична.
  • Аналог на ≤ за нарастване на функции.
Изпитен въпрос

Дефинирайте O-нотацията. Докажете, че \(n^2 + 5n = O(n^2)\).

Ω-нотация — Долна асимптотична граница

3
Точна дефиниция
Определение — Ω-нотация (Big-Omega)

Нека \(g: \mathbb{N} \to \mathbb{R}^+\). Дефинираме:

\[\Omega(g(n)) = \{ f \mid \exists c > 0,\, n_0 \in \mathbb{N} : \forall n \geq n_0 : f(n) \geq c \cdot g(n) \}\]

Казваме \(f(n) = \Omega(g(n))\). \(g\) е асимптотична долна граница на \(f\).

Теорема 18: Транспониране (дуалност)

\(f(n) = O(g(n))\) тогава и само тогава когато \(g(n) = \Omega(f(n))\).

Примери
  • \(n^2 = \Omega(n)\) — с \(c=1, n_0=1\)
  • \(n \log n = \Omega(n)\)
  • Heapsort: \(T(n) = \Omega(n \log n)\) — никой сортиращ алгоритъм, базиран на сравнения, не може да работи по-бързо.
За изпит
  • Ω е долна граница — аналог на ≥.
  • Дуалност: \(f = O(g) \Leftrightarrow g = \Omega(f)\).
  • Ω се използва за долни граници на задачи, не само на алгоритми.
Изпитен въпрос

Дефинирайте Ω-нотацията. Докажете дуалността: \(f = O(g) \Leftrightarrow g = \Omega(f)\).

o и ω нотации — Строги граници

4
Точни дефиниции
Определение — o-нотация (Little-o)

Нека \(g: \mathbb{N} \to \mathbb{R}^+\). Дефинираме:

\[o(g(n)) = \{ f \mid \forall c > 0,\, \exists n_0 \in \mathbb{N} : \forall n \geq n_0 : f(n) < c \cdot g(n) \}\]

Еквивалентно: \(f(n) = o(g(n)) \Leftrightarrow \lim_{n\to\infty} \dfrac{f(n)}{g(n)} = 0\).

Определение — ω-нотация (Little-omega)

\[\omega(g(n)) = \{ f \mid \forall c > 0,\, \exists n_0 \in \mathbb{N} : \forall n \geq n_0 : f(n) > c \cdot g(n) \}\]

Еквивалентно: \(f(n) = \omega(g(n)) \Leftrightarrow \lim_{n\to\infty} \dfrac{f(n)}{g(n)} = +\infty\).

Разлика между O и o, Ω и ω
НотацияКвантор за cИнтуицияАналог
\(O(g)\)\(\exists c > 0\)f расте не по-бързо от g\(\leq\)
\(o(g)\)\(\forall c > 0\)f расте строго по-бавно от g\(<\)
\(\Omega(g)\)\(\exists c > 0\)f расте не по-бавно от g\(\geq\)
\(\omega(g)\)\(\forall c > 0\)f расте строго по-бързо от g\(>\)
\(\Theta(g)\)\(\exists c_1, c_2 > 0\)f расте точно като g\(=\)
Теореми 19, 20: Свойства
  • Дуалност: \(f = o(g) \Leftrightarrow g = \omega(f)\)
  • Ако \(f = o(g)\), то \(f = O(g)\), но не обратното.
  • Ако \(f = o(g)\), то НЕ е вярно, че \(f = \Theta(g)\).
Примери
  • \(n = o(n^2)\): \(\lim n/n^2 = 0\) ✓
  • \(n^2 = \omega(n)\): \(\lim n^2/n = \infty\) ✓
  • \(n \log n \neq o(n^2)\): но \(n \log n = o(n^2)\) ✓ (понеже \(n \log n / n^2 = \log n / n \to 0\))
За изпит
  • o: квантор за c е ∀ (за всяко c, от достатъчно голямо n). O: ∃ c.
  • \(f = o(g) \Rightarrow \lim f/g = 0\). \(f = \omega(g) \Rightarrow \lim f/g = \infty\).
  • o и ω са асиметрични (нерефлексивни) — \(f \neq o(f)\) и \(f \neq \omega(f)\).
Изпитен въпрос

Дефинирайте o и ω нотациите. Каква е разликата между O и o? Докажете чрез граница на редица.

Свойства на нотациите — Теореми 21-30

5
Основни свойства
Теорема 21: Транзитивност

За всяко \(* \in \{O, \Omega, \Theta, o, \omega\}\): Ако \(f = *(g)\) и \(g = *(h)\), то \(f = *(h)\).

Теорема 22: Рефлексивност (за O, Ω, Θ)

\(f = O(f)\), \(f = \Omega(f)\), \(f = \Theta(f)\). (Но \(f \neq o(f)\) и \(f \neq \omega(f)\).)

Теорема 23: Симетричност на Θ

\(f = \Theta(g) \Leftrightarrow g = \Theta(f)\).

Теорема 24: Сума

Ако \(f_1 = O(g_1)\) и \(f_2 = O(g_2)\), то \(f_1 + f_2 = O(\max(g_1, g_2))\).

По-специфично: \(f_1 + f_2 = O(g_1 + g_2)\).

Следствие: В полинома \(a_k n^k + a_{k-1} n^{k-1} + \ldots + a_0\), доминира само най-старшият член: \(= \Theta(n^k)\).

Теорема 25: Произведение

Ако \(f_1 = O(g_1)\) и \(f_2 = O(g_2)\), то \(f_1 \cdot f_2 = O(g_1 \cdot g_2)\).

Теорема 26: Константен множител

За всяка константа \(c > 0\): \(c \cdot f(n) = \Theta(f(n))\). С други думи, константните множители се пренебрегват в асимптотичния анализ.

Теорема 27: Полиноми и логаритми

За всяко \(a > 1\) и \(b > 0\): \((\log n)^b = o(n^a)\).

Следователно: полиномите надрастват логаритмите.

Теорема 28: Полиноми и експоненти

За всяко \(a > 1\) и \(b > 0\): \(n^b = o(a^n)\).

Следователно: експонентите надрастват полиномите.

Теорема 29: Приближение на Stirling

\(n! = \sqrt{2\pi n} \left(\dfrac{n}{e}\right)^n \left(1 + \Theta\!\left(\dfrac{1}{n}\right)\right)\)

Следствие: \(\log(n!) = \Theta(n \log n)\).

Теорема 30: Хармоничен ред

\(H_n = \sum_{k=1}^{n} \dfrac{1}{k} = \Theta(\log n)\).

По-прецизно: \(H_n = \ln n + \gamma + O(1/n)\), където \(\gamma \approx 0.5772\) е константата на Euler-Mascheroni.

Наредба на стандартните функции
По-бавно нарастване → по-бързо нарастване: 1 ≺ log log n ≺ log n ≺ (log n)² ≺ √n ≺ n ≺ ≺ n log n ≺ n² ≺ n³ ≺ ... ≺ nᵏ ≺ ... ≺ ≺ 2^(√n) ≺ 2ⁿ ≺ n·2ⁿ ≺ n! ≺ nⁿ (≺ означава o, т.е. строго по-бавно нарастване)
За изпит
  • Наизуст: Теореми 24 (сума), 26 (константа), 27 (log vs poly), 28 (poly vs exp), 29 (Stirling).
  • В сума — само най-бързо растящият член има значение.
  • Логаритмите винаги отстъпват на полиномите, полиномите — на експонентите.
Изпитен въпрос

Докажете Теорема 24 (сума) или Теорема 26 (константа). Наредете функциите: \(n!, 2^n, n^2, n\log n, \log n, \sqrt{n}\).

Рекурентни уравнения — дефиниция и класификации

7
Точна дефиниция
Рекурентно уравнение

Рекурентно уравнение е уравнение, в което вляво от "=" стои \(T(n)\) (или \(a_k\)), а вдясно е валиден израз, съдържащ поне едно появяване на \(T\) с по-малки аргументи: например \(T(n/2)\) или \(T(n-1)\). Допълнено с начални условия, то определя еднозначно числова редица.

Класификации
По функцията на намаляване
  • Изваждане на константа: \(T(n) = 2T(n-1) + 1\) — типично за Пълно Изчерпване.
  • Деление на константа: \(T(n) = 2T(n/2) + n\) — типично за Разделяй-и-Владей.
Линейни vs нелинейни
  • Линейно: \(T(n) = c_1 T(n-1) + c_2 T(n-2) + f(n)\)
  • Нелинейно: \(T(n) = T(n-1) \cdot T(n-2) + 1\) — не се покрива от стандартни методи.
Хомогенни vs нехомогенни
  • Хомогенно: \(T(n) = c_1 T(n-1) + c_2 T(n-2)\) — само от T-а вдясно.
  • Нехомогенно: \(T(n) = c_1 T(n-1) + c_2 T(n-2) + f(n)\) — има допълнителен член \(f(n)\).
Примери
  • Fibonacci: \(F_n = F_{n-1} + F_{n-2}\) — линейно, хомогенно, константни коефициенти, 2-ри ред.
  • Mergesort: \(T(n) = 2T(n/2) + n\) — линейно, нехомогенно, деление.
  • SELECT: \(T(n) = T(n/5) + T(7n/10) + n\) — нехомогенно, деление.
За изпит
  • Рекурентното уравнение е различен обект от алгоритъма, чиято сложност описва!
  • Ред на уравнение = най-голямото число, което се изважда вдясно.
  • За Разделяй-и-Владей: функцията на намаляване е деление.
Изпитен въпрос

Дефинирайте рекурентно уравнение. Класифицирайте \(T(n) = 3T(n-1) + 4T(n-2) + n\) по ред, линейност, хомогенност и вид на намаляването.

Метод на развиването (Unfolding)

8
Описание на метода

Методът на развиването (на английски "unfolding" или "iteration") е директен начин за решаване на рекурентно уравнение: заместваме рекурентно докато достигнем базовия случай, след което сумираме.

Задача 26: Решение на T(n) = T(n-1) + n
T(n) = T(n-1) + n = T(n-2) + (n-1) + n = T(n-3) + (n-2) + (n-1) + n = ... = T(1) + 2 + 3 + ... + n = 1 + n(n+1)/2 - 1 = n(n+1)/2 = Θ(n²)
Задача 27: Решение на T(n) = 2T(n/2) + n
T(n) = 2T(n/2) + n = 2[2T(n/4) + n/2] + n = 4T(n/4) + n + n = 4[2T(n/8) + n/4] + 2n = 8T(n/8) + n + n + n = ... = 2^k · T(n/2^k) + k·n При k = log₂n: n/2^k = 1, T(1) = 1 = n · T(1) + n·log₂n = n + n·log₂n = Θ(n log n)
За изпит
  • Развиваме k пъти, изразяваме T(n) чрез T(n/b^k) или T(n-k).
  • При деление: избираме k = log_b(n), при което T(1) или T(константа) е базовия случай.
  • Резултатът е точна формула, не само асимптотика!
Изпитен въпрос

Решете чрез развиване: \(T(n) = T(n/3) + n\), \(T(1) = 1\).

Дърво на рекурсията (Recursion Tree)

9
Описание на метода

Дървото на рекурсията е визуален метод: разгръщаме рекурентното уравнение като дърво, изчисляваме работата на всяко ниво и сумираме по всички нива.

Дърво на рекурсията за T(n) = 2T(n/2) + n
Ниво 0: n = n Ниво 1: n/2 + n/2 = n Ниво 2: n/4+n/4+n/4+n/4 = n Ниво 3: n/8+n/8+... = n ... Ниво log₂n: 1+1+1+...+1 (n листа) = n Брой нива: log₂n + 1 Работа на всяко ниво: n Общо: n · (log₂n + 1) = Θ(n log n)
Дърво на рекурсията за T(n) = T(n/3) + T(2n/3) + n
Ниво 0: n = n Ниво 1: n/3 + 2n/3 = n Ниво 2: n/9+2n/9+2n/9+4n/9 = n ... Левият клон стига до листо след log₃n нива. Десният клон стига до листо след log_{3/2}n нива. Т.е. дървото има между log₃n и log_{3/2}n нива. Общо ≈ cn·log n → Θ(n log n) [по Мастър теорема: a=2,b неизвестно,f(n)=n]
За изпит
  • Изчислете работата на всяко ниво (сума на всички викания на това ниво).
  • Изчислете броя нива (дълбочина на дървото).
  • Сумирайте.
  • Методът дава добра интуиция, но резултатът трябва да се провери по индукция!
Изпитен въпрос

Начертайте дърво на рекурсията за \(T(n) = 3T(n/4) + n\). Кое е решението?

Мастър теорема

10
Точна формулировка
Мастър теорема (Master Theorem)

Нека \(T(n) = aT(n/b) + f(n)\), където \(a \geq 1\), \(b > 1\) са константи и \(f(n) \geq 0\). Нека \(p = \log_b a\). Тогава:

  • Случай 1: Ако \(f(n) = O(n^{p-\varepsilon})\) за някакво \(\varepsilon > 0\), то \(T(n) = \Theta(n^p)\).
  • Случай 2: Ако \(f(n) = \Theta(n^p)\), то \(T(n) = \Theta(n^p \log n)\).
  • Случай 3: Ако \(f(n) = \Omega(n^{p+\varepsilon})\) за някакво \(\varepsilon > 0\) и ако \(af(n/b) \leq cf(n)\) за константа \(c < 1\) и достатъчно голямо \(n\), то \(T(n) = \Theta(f(n))\).
Интуиция за трите случая
Сравняваме f(n) с "работата на листата" = n^p = n^{log_b(a)} Случай 1: Листата доминират (f(n) расте по-бавно) → T(n) = Θ(n^p) Случай 2: Всяко ниво работи еднакво → T(n) = Θ(n^p · log n) Случай 3: Коренът (f(n)) доминира → T(n) = Θ(f(n))
Примери за прилагане
Уравнениеa, b, pf(n)СлучайРешение
\(T=2T(n/2)+n\)a=2,b=2,p=1\(n=\Theta(n^1)\)2\(\Theta(n\log n)\)
\(T=4T(n/2)+n\)a=4,b=2,p=2\(n=O(n^{2-1})\)1\(\Theta(n^2)\)
\(T=4T(n/2)+n^3\)a=4,b=2,p=2\(n^3=\Omega(n^{2+1})\)3\(\Theta(n^3)\)
\(T=2T(n/2)+n\log n\)a=2,b=2,p=1\(n\log n\) — не покрива2 (разширен)\(\Theta(n\log^2 n)\)

Внимание: Мастър теоремата не покрива всички случаи! Например \(T(n) = T(n/2) + n/\log n\) е между случай 1 и случай 2 и теоремата не е директно приложима.

За изпит
  • Изчисли \(p = \log_b a\). Сравни \(f(n)\) с \(n^p\).
  • Трите случая: f расте по-бавно (Сл.1), равно (Сл.2), по-бързо (Сл.3) от \(n^p\).
  • В Случай 3 трябва регулярността: \(af(n/b) \leq cf(n)\).
  • Ако случаят не е ясен — опитай развиване или дърво на рекурсията.

Теореми 38 и 39 — Решение по индукция

11
Точни формулировки
Теорема 38: Метод на индукцията за T(n) = T(n-1) + f(n)

Нека \(T(1) = c\) и \(T(n) = T(n-1) + f(n)\) за \(n > 1\). Тогава:

\[T(n) = c + \sum_{k=2}^{n} f(k)\]

Доказателство: По индукция по n. База: T(1)=c. Стъпка: T(n) = T(n-1) + f(n) = c + \sum_{k=2}^{n-1} f(k) + f(n) = c + \sum_{k=2}^{n} f(k).

Теорема 39: Метод за T(n) = T(αn) + T(βn) + cn с α+β < 1

Нека \(T(n) = T(\alpha n) + T(\beta n) + cn\), където \(\alpha, \beta > 0\) и \(\alpha + \beta < 1\). Тогава:

\[T(n) = \Theta(n)\]

Доказателство по индукция: Предполагаме \(T(n) \leq dn\) за константа \(d > c/(1-\alpha-\beta)\). Тогава:

\[T(n) = T(\alpha n) + T(\beta n) + cn \leq d\alpha n + d\beta n + cn = dn(\alpha + \beta) + cn\]

Ако \(d(\alpha+\beta) + c \leq d\), тоест \(d \geq c/(1-\alpha-\beta)\), получаваме \(T(n) \leq dn\). Dolna граница се доказва аналогично.

Приложение на Теорема 39
SELECT: T(n) = T(n/5) + T(7n/10) + n

Имаме \(\alpha = 1/5\), \(\beta = 7/10\), \(\alpha + \beta = 9/10 < 1\).

По Теорема 39: \(T(n) = \Theta(n)\).

Константата: \(d \geq c/(1-9/10) = 10c\). Ако \(c=1\) (единица работа на ниво), то \(d = 10\).

Решение по индукция — обща схема
Стъпки при доказателство по индукция
  1. Предположи вида на решението (например \(T(n) \leq cn\)).
  2. База: Провери за малки n.
  3. Стъпка: Замести \(T\) вдясно с предположението и провери дали получаваш \(\leq cn\) вляво.
  4. Избери константите така, че да работи.
