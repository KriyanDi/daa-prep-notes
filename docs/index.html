Преподавателят цени точния математически запис. Учи дефинициите дословно — особено Определение 1 (изчислителна задача), Θ-нотация и Теорема 93 (МПД).

Инварианти на цикли

Почти всяко доказателство за коректност на итеративен алгоритъм изисква инвариант. Учи го за: Insertion Sort, Heapify, BFS, Dijkstra, Build Heap.

Редукциите (Лекция 15)

Знай веригата: SAT →p 3SAT →p Vertex Cover →p Independent Set →p Clique. И: Vertex Cover →p Dominating Set.

⏱️

Сложности наизуст

Insertion Sort: Θ(n²). Heapsort: Θ(n log n). Build Heap: Θ(n). Mergesort: Θ(n log n). Quicksort: O(n²) worst, O(n log n) avg. Dijkstra: O((V+E) log V). Floyd-Warshall: Θ(V³).

⚠️

Чести грешки

Не бърказадача и алгоритъм. Dijkstra изисква неотрицателни тегла. Build Heap е Θ(n), не Θ(n log n). P ⊆ NP, но не знаем дали P = NP.
