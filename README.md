## Конспект — обхванати теми

### Фундамент
1. **Алгоритми и изчислителни задачи** — видове задачи, функция съответна на задача, коректност, инварианти на цикъла
2. **Асимптотичен анализ** — нотации Θ, O, Ω, o, ω; теореми за свойства; рекурентни уравнения; Master Theorem

### Сортиране
3. **Insertion Sort** — двоично търсене, стабилност, доказателство за коректност
4. **Heapsort** — двоична пирамида, Build Heap Θ(n), приоритетни опашки
5. **Разделяй-и-Владей** — SELECT (медиана на медианите), Mergesort, Quicksort; средна сложност
6. **Counting Sort** — стабилност, лема 33 & 34, теорема 75

### Графи
7. **BFS** — ниво по ниво, разстояния, дърво на обхождане
8. **DFS + Топосортиране + Kosaraju** — интервали, скобова структура, силно свързани компоненти
9. **Срязващи върхове + MST** — Prim, Kruskal, Union-Find с Union by Rank

### Най-кратки пътища & Динамично програмиране
10. **Dijkstra & Bellman-Ford** — релаксация, отрицателни цикли
11. **ДП I** — Floyd-Warshall, Matrix-Chain, 0-1 Knapsack, Interval Scheduling
12. **ДП II** — LIS, LCS, Edit Distance, Weighted Independent Set на дървета

### Сложност
13. **Долни граници** — Ω(n log n) за сортиране, adversary argument, медиана
14. **P, NP, NP-пълнота** — машини на Тюринг, Karp редукции, класове NP-c и NP-h
15. **SAT, 3-SAT, Cook** — верига от NP-пълни редукции до Vertex Cover, Clique, Dominating Set

## Ключови доказателства за изпита

Изпитът изисква формални доказателства. Ето пълния списък:

**Инварианти на цикъла**
- Insertion Sort, Mergesort, Quicksort, Counting Sort (лема 33, 34, теорема 75), Двоично търсене

**Асимптотичен анализ**
- Свойства на нотация Θ (Теорема 16) · Master Theorem · Build Heap е Θ(n)

**Графи**
- Скобова структура на DFS интервали (Теорема 78) · Коректност на Kosaraju (Теорема 87, 88) · Теорема 89, 90 (срязващи върхове) · Коректност на Dijkstra (Теорема 100, 101) · Floyd-Warshall

**Долни граници**
- Ω(n log n) за сортиране (Теорема 112) · Element Uniqueness · намиране на максимум и медиана

**NP-пълнота**
- Теорема на Cook (SAT е NP-пълна, Теорема 135) · Редукции 10, 14, 15, 16, 17
